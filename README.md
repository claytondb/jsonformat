# JSON Format

A free online JSON beautifier, validator, and minifier for developers.

## Features

- **Format/Beautify** - Pretty-print JSON with proper indentation
- **Minify** - Compress JSON by removing whitespace
- **Validate** - Check JSON syntax and report errors
- **Syntax Highlighting** - Color-coded JSON display
- **Sample Data** - Quick-load example JSON

## Usage

Open `index.html` in any modern browser. No installation required.

1. Paste or type JSON in the input area
2. Click Format, Minify, or Validate
3. Copy the result

## Use Cases

- Pretty-printing API responses
- Validating JSON configuration files
- Compressing JSON for production
- Debugging malformed JSON

## Keyboard Shortcuts

- `Ctrl+Enter` - Format
- `Ctrl+Shift+M` - Minify
- `Ctrl+V` - Paste (browser built-in)

## Tech

- Pure HTML/CSS/JavaScript
- No external dependencies
- Works offline
- Responsive design
